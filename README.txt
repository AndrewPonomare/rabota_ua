Проект загружен на хостинг Netlify https://sharp-allen-591d33.netlify.app/
Контент вакансий взял с API rabota.ua. API не является публичным но если делать запрос с помощью Node.js то приходи ответ. 
Если создать свой сервер на Node.js то можно пользоваться полноценным функционалом API (Например реализовать поиск по вакансиям)
Для запуска демо используем команду "npm run start".
В целом было интересно разрабатывать этот проект, в дальнейшем хочу перенести всё на React.js
